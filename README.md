# Sistema-de-Monitoramento-Ambiental-Rio-Tiet-
O software simula uma ferramenta de comunicação em rede para a Secretaria de Estado do Meio Ambiente. O objetivo é permitir que inspetores distribuídos ao longo do Rio Tietê (desde Salesópolis até a Grande São Paulo) enviem informações online sobre atividades industriais e níveis de poluição para uma central de controle.
